Detailed Description

This are all the methods that draw directly on the Window. With them you can draw a bunch of pretty standard figures that will allow you to create more complex ones.

void Window::arc ( float  cx,
float  cy,
float  r,
float  start_theta,
float  delta_theta,
RGBColor  color,
float  thickness = 1.0 
)

Draws an arc given its center's coordinates, its radius and the angle it describes. The angle is represented as the start point of the angle start_theta, and the angle increase delta_theta. This angle can be expressed in radians or degrees depending on the Window configuration (degrees by default). For example this code:

#include "WindowAddon.h"
int main(int argc, char **argv){
Window w(201, 201); //White background color and black drawing color by default
w.arc(101, 75, 50, 0, 180, BLACK); //thickness 1.0 by default
w.waitWindowCloseButton();
return 0;
}

Would generate this output:

Parameters
cxX coordinate of the center of the arc.
cyY coordinate of the center of the arc.
rRadius of the arc.
start_thetaStart point of the angle.
delta_thetaAngle increase.
colorColor used to draw the figure.
thicknessThickness of the line. Its value is 1.0 by default.
